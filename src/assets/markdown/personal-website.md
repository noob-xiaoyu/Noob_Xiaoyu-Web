# 个人网站项目

## 项目概述
这是一个完全响应式的个人主页和作品集网站，展示我的技术能力和项目经验。

## 技术栈
- **前端框架**: Vue 3 + Composition API
- **构建工具**: Vite
- **UI 组件库**: Element Plus
- **路由管理**: Vue Router 4
- **状态管理**: Pinia（可选）
- **样式方案**: CSS3 + Flexbox/Grid
- **动画效果**: CSS Transitions + Vue Transition

## 核心功能

### 🎨 视觉效果
- 动态视频背景，支持暂停/播放控制
- 黑白主题一键切换
- 响应式设计，适配各种屏幕尺寸
- 平滑的页面过渡动画

### ⚡ 性能优化
- 按需加载组件
- 图片懒加载
- 代码分割和异步路由
- 优化首次加载时间

### 📱 用户体验
- 移动端友好设计
- 键盘导航支持
- 页面加载进度指示
- SEO 基础优化
